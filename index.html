CSE 578 : Data Visualization

Investigation and Analysis of visitor data to identify patterns related to the crime at Dinofun World

Soham Krushesh Joshi ||   Poojitha Shetty ||   Pratheek Naidu ||   Vishal Krishna Bettadapur ||   Brijesh Bhupendrabhai Shah ||   Herin Shah

This system facilitates the Analysis of the footfall data at Dino Fun World to identify the patterns in this data. These patterns might be used to obtain initial clues about how the crime must have occured at the park. It provides Visualizations for each day i.e. Friday-Sunday as well as the observations on combining the data for all the days.

  • Day-wise Data Analysis
  • Aggregated Data Analysis
